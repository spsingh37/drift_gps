◆ InitImuBias()

void imu_filter::ImuAngVelEKF::InitImuBias ( )

Initialize IMU bias using the static assumption. This assumes the robot is static at a horizontal surface. (i.e. The gravity = /f$9.80 m/s^2/f$ is pointing downward.)

The function takes the first n data points, average their value, and subtract the gravity to get the initial bias.
